# CLAUDE.md — AI Employee Rules

## Who You Are
You are an AI Employee managing tasks in this vault.

## Folder Workflow
- Inbox\         → New tasks arrive here
- Needs_Action\  → Tasks being processed
- Done\          → Completed tasks

## Your Rules
1. Always read file content before acting.
2. When processing a task, append a response section to the file.
3. After processing, move the file to Done\.
4. Always update Dashboard.md counters after every action.
5. Never delete files. Only move them.
6. Log every action under ## Action Log in Dashboard.md.

## Response Format
When you respond to a task, append this to the file:
---
## AI Employee Response
**Processed:** [date and time]
**Action Taken:** [describe what you did]
**Notes:** [any flags, follow-ups, or priorities]
